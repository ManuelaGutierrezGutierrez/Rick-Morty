<h1 align="left"> Rick & Morty </h1>


Web with the characters of Rick & Morty. 


## Stack

* React with useState
* SASS
* Rick & Morty API: https://rickandmortyapi.com/

## Functionalities

* Pagination
* Search bar
* Responsive
